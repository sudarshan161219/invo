# Free Invoice & Document Generator

A **free, browser-based document generator** for creating professional **Invoices, Quotations, Estimates, Receipts, and Credit Notes**.

This is a **frontend-only portfolio project**. All data is stored **locally in the browser** using local storage.  
No backend, no accounts, no tracking.

---

## Features

- Create multiple document types:
  - Invoice
  - Tax Invoice
  - Quotation
  - Estimate
  - Receipt
  - Credit Note
- Works fully in the browser
- No login or signup required
- Offline-friendly after first load
- Client and shipping details
- Multiple line items
- Tax and discount support
- Multiple currencies
- Logo and signature support
- Multiple payment methods
- Clean, print-ready document layout
